
# React, Socket.io & Chakra ui Chat App

Socket.io ve React ile yapılan basit bir gerçek zamanlı chat web uygulaması.

## Bilgisayarınızda Çalıştırın

İki proje için gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  cd server
```

```bash
  npm run start
```

Uygulamayı çalıştırın

```bash
  cd app
```
```bash
  npm run start
```
